## Usage in Tailwind Classes
Use these custom classes:
- `bg-primary`, `bg-primary-dark`, `bg-primary-light`, `bg-primary-darker`
- `text-primary`, `text-primary-dark`, `text-primary-darker`
- `bg-secondary`, `bg-secondary-dark`, `bg-secondary-light`
- `text-secondary`, `text-secondary-dark`
- `bg-accent`, `bg-accent-dark`, `bg-accent-light`
- `text-accent`, `text-accent-dark`
- `focus:ring-primary` for focus states
- `gradient-primary` for gradients
